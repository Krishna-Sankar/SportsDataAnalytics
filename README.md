SportsDataAnalytics
===================

This is a growing collection of sports analytics resources, including code, data sets & statistics, visualizations, metrics, predictive analytics, machine learning, as well as other data-driven sports research, labs, conferences, papers, pioneers & news about baseball, football, basketball, soccer, hockey, tennis...for pro, college & highschool level, men & women's sports.*
